Lab Desarrollo de Software

Este proyecto es un laboratorio para aprender y practicar los conceptos fundamentales del desarrollo de software, con un enfoque especial en el control de versiones y ramas utilizando Git y GitHub.

## Descripción del Proyecto

El objetivo de este repositorio es servir como un espacio de práctica para un equipo de desarrolladores. A través de la creación de diferentes ramas (feature/*), se simula un flujo de trabajo colaborativo donde cada participante aporta cambios, los documenta y los integra en una rama principal (develop) antes de ser fusionados a la rama main.

El proyecto incluye archivos de configuración, documentación inicial y pruebas, reflejando una estructura básica de un proyecto de software.

## Lista de Comandos Utilizados y su Propósito

Dado que todo el trabajo se gestiona desde la consola, aquí hay una lista de los comandos de Git más comunes que se utilizan en este proyecto:

- git clone <url>: Clona el repositorio remoto a tu máquina local para empezar a trabajar.
- git branch: Muestra todas las ramas locales y resalta la rama actual.
- git checkout -b <nombre-de-la-rama>: Crea una nueva rama y cambia a ella de inmediato. Usado para empezar una nueva funcionalidad (ej. feature_Juan_Ramos).
- git add .: Añade todos los archivos modificados o nuevos al área de preparación (staging area) para el próximo commit.
- git commit -m "Mensaje descriptivo": Guarda una instantánea de los cambios preparados con un mensaje que explica qué se ha hecho.
- git push origin <nombre-de-la-rama>: Sube los commits de tu rama local al repositorio remoto (GitHub), permitiendo a otros ver tus cambios y crear Pull Requests.
- git pull: Actualiza tu rama local con los últimos cambios del repositorio remoto. Esencial para mantener tu trabajo sincronizado.
- git merge <nombre-de-la-rama-a-fusionar>: Fusiona los cambios de una rama (ej. una rama de feature) en la rama actual (ej. develop).
- git log --oneline --graph --decorate: Muestra un historial de commits de forma visual y resumida, ideal para ver la estructura de ramas y fusiones.
