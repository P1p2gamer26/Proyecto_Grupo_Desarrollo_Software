# Overview

Este proyecto forma parte de un taller práctico de desarrollo de software cuyo objetivo principal es aprender y practicar el uso de Git y el control de versiones.

Durante el taller se trabajan principalmente:
- Creación de carpetas y archivos
- Organización básica de un proyecto
- Simulación de flujos de trabajo colaborativos y control de versiones mediante pull requests y manejo de ramas
